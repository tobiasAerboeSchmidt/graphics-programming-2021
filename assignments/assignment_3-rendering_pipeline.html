Assignment 3 - The rendering pipeline

Learning objectives
  • Outline and describe the steps in the rendering pipeline.
  • Describe the role of each step, their inputs and outputs.
  • Reason about the order of steps in the rendering pipeline.
  • Describe how clipping and culling affect render performance.
  • Describe coordinate spaces involved in the rendering pipeline.
(This is an individual assignment. Once again, we will use peergrading for feedback.)
You will have two weeks to work on this assignment.
Assignment 3
Describe the rendering pipeline
Write a short paper (1~2 pages), in which you describe all the steps in the rendering pipeline, as presented in the figure below (you can skip the tesselation and geometry shaders). You can use the OpenGL pipeline documentation and the Software Renderer Library (SRL, the software renderer that I provide under the name "exercise 7"), which follows the same order of operation as OpenGL, as references. You can also check the related videos on learnit/youtube (the OneLoneCoder videos), but mind that the order of operations is not the same as in OpenGL.

For each step, explain:
  • what it does;
  • why it is necessary;
  • why do you think it happens at this point in the pipeline (and not before or after);
  • what is the input information;
  • and what is the generated output.
Also describe how the following coordinate spaces (particular charactersitics, what and where it is used for, etc.):
  • eye/camera space;
  • Clipping space;
  • Normalized Device Coordinates (NDC);
  • Screen space.
