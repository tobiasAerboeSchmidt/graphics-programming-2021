Exercise 5 - Cameras and projections

Learning objectives
  • Use the glm helper functions to create a perspective camera, defining position, direction and projection parameters
  • Implement a first person camera controller (glfw for input).
-We will be using the library glm to facilitate the manipulation of matrices and vectors. The tutorial https://learnopengl.com/Getting-started/Transformations might prove helpful. You also have the option to use the glmutils library, which is included in the exercise.
-The tutorial https://learnopengl.com/Getting-started/Camera can help to understand the camera movement parameters.
Exercise 5.1
Perspective camera
Implement a perspective camera in the trackball application:
  • Using the glm::lookat function, create a transformation matrix that maps points in the world space to camera space. You can set the parameters:
    • position to (0,0,2)
    • forward to (0,0,-1)
  • Using the glm::perspectiveFov function, create a projection matrix with the parameters:
    • field of view = 70 degrees (notice that perspectiveFov expects the angle in radians);
    • Width and height should be the same as the program window size;
    • Near plane = .01f;
    • Far plane = 10.0f;
  • Left multiply the view matrix with the projection matrix, you should now be able to visualize Exercise 4.3/4.4 using a perspective projection instead of ortographic projection;
  • Notice that the depth range is now set to glDepthRange(-1,1), that is, a left handed coordinate system. This is because the default openGL's NDC is in a left handed coordinate system, and the default perspective camera projection we have seen in class (equivalent to glm::perspectiveFov) converts from a right handed coordinate system in the application side to a left handed coordinate system in the rendering side;
Exercise 5.2
First person camera controller
Implement a first person camera controller and get the view matrix using either the glm::lookat funtion or a composition of translation and rotation matrices:
  • Using the WASD or arrow keys to move forward, backward, left and right.
  • Use the mouse movements to rotate the camera. You can start implementing rotation so that you can look to the sides, and then extend so that you can look up and down.
  • (OPTIONAL) Use the left mouse button to throw airplanes from your point of view.
