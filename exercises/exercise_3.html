ITU Graphics Programming

Exercise 3 - Interaction and geometric transformations

Learning objectives
  • Manipulate vectors and transformation matrices using GLM;
  • Use transformations to create simple scenes;
  • Use model transformations for controlling virtual objects in the world coordinate frame.
-We will be using the GLM library to facilitate the manipulation of matrices and vectors. The tutorial https://learnopengl.com/Getting-started/Transformations might prove helpful. You also have a wrapper class called "glmUtils" that you can use, it facilitates the creation of certain transformation matrices.
Exercise 3.1
 Creating a plane - setup
  • The base code for this exercise has depth testing on. That is, a depth buffer (aka z-buffer) that prevents occluded fragments from being shown in the screen. Search for the string NEW! to see how this is activated in OpenGL.
  • Introduce one mat4 uniform in the vertex shader called 'model'. The input vertex in the vertex shader should be transformed by the model matrix.
    • Set the model uniform to the identity matrix in the drawPlane function – before drawing both the plane body and wing. The model matrix allows us to later change the position, scale and rotation of the individual parts.
    • The completion of this step should not change the rendered image on the left, and should not throw any errors.
Exercise 3.2
Add wings to plane
  • Use translation and scale to add a left wing as well as two tail wings by modifying the drawPlane function.
    • 'drawSceneObject(planeWing);' has to be called four times in drawPlane but before each call the 'model' uniform has to be set.
    • Hint: mirroring can be achieved by scaling one axis with a negative value.
Exercise 3.3
Add propeller to plane
  • The code contains a propeller.
  • Create and initialize propeller SceneObject in the setup function (repeat the steps used to load the body and wing of the plane).
  • Position and scale the propeller correctly to the top of the plane. This means that the propeller should be scaled by a factor of 0.5 and translated.
  • Make the propeller rotate.
Exercise 3.4
Controlling the plane
  • Change the model matrix such that the whole plane scale is reduced to 1/10 of its original size.
  • Introduce the variables planePos (Vec2) and planeRotation (number).
  • The A and D keys should be used to steer the plane (by gradually rotating the plane around the z axis.
  • Make the plane move in its forward direction at a constant speed.
  • If the A key is pressed the plane should also rotate -45 degrees around its local y axis. And if D key is pressed, rotate 45 degrees around its local y axis.
  • If the plane flies out the visible canvas, it should be “wrapped around” (if it flies out of the left side it should appear from the right side).
